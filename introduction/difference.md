# 引擎差异

OpenSearch是基于 Elasticsearch 7.10.2 版本分叉而来，全程采用 Apache 2.0 完全开源协议，无任何商业使用限制，支持企业自由使用、修改、分发源代码，彻底规避授权风险。项目由开源社区自主治理，功能迭代完全开放，是企业实现技术自主可控的理想选择。UCloud CSS OpenSearch 引擎深度适配开源最新版本，持续同步社区特性，为用户提供无锁定的开源搜索服务。

## 功能特性与生态工具的差异

Elasticsearch 引擎功能体系经过十余年市场验证，覆盖全文检索、模糊查询、聚合分析等全场景能力，生态工具链成熟完善：

○数据采集：原生支持 Logstash，可无缝对接 Kafka、对象存储等数十种异构数据源；

○可视化分析：配套全托管 Kibana 服务，支持一站式数据探索、仪表盘搭建、运维监控；

○生态扩展：全球海量开源插件、第三方工具与解决方案，适配各类企业级业务需求。


OpenSearch 引擎完整继承 Elasticsearch 核心检索能力，同时持续迭代特色增强功能，生态工具适配云原生架构：
○核心能力：兼容 Elasticsearch 7.10.2 版本 API，支持存量 ES 业务平滑迁移，新增向量检索、存算分离、冷热分层等云原生特性；
○数据采集：提供 Data Prepper 作为 Logstash 替代方案，更适配云原生数据管道场景；
○可视化分析：配套 OpenSearch Dashboards，功能与 Kibana 高度对齐，同时新增 AI 搜索、安全审计等专属能力；
○开源扩展：所有功能完全开源，无商业功能限制，支持企业自由定制化开发。

## Opensearch的优势

OpenSearch 引擎基于云原生架构持续优化，在存算分离、弹性伸缩、冷热数据分层等云托管场景针对性增强，同时在向量检索、AI 搜索等新兴场景持续迭代性能，适配云原生业务的弹性扩缩容需求，更贴合云服务的使用场景。UCloud CSS OpenSearch 引擎深度适配云原生架构，支持分钟级部署、无损弹性扩缩容，满足业务快速迭代需求。

OpenSearch 引擎开源原生集成完整安全能力，无功能阉割，支持细粒度访问控制、审计日志、数据加密、身份认证等，同时开源社区持续迭代安全特性，所有安全能力完全开源，无商业授权限制，适配企业自主可控的安全合规需求。
