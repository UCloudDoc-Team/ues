# 索引模板管理

索引模板用于自动化配置新索引。当创建符合特定命名模式的新索引时，UES会自动应用模板中预定义的配置，确保索引配置标准化，避免手动配置错误，提升运维效率。本文介绍如何管理索引模板。

## 创建索引模板

![image](/images/operate/index_template_1.png)

1. 在索引管理页面，选择索引模板页签，然后单击创建索引模板。
2. 在创建索引模板面板中，配置以下参数，然后单击确定。

| **参数**         | **说明**                                   |
| ---------------- | ------------------------------------------ |
| 模板名称         | 设置索引模板的名称。限制如下：不能以数字、下划线（_）开头。长度为 1~128 个字符。只能包含小写字母、数字、下划线（_）和中划线（-）。 |
| 优先级           | 设置索引模板的优先级。创建索引的名称与多个模板匹配时，优先级最高的模板应用于该索引。建议值 1~1000。 |
| 索引模式         | 设置索引模式。多个索引模式之间用英文逗号（,）或回车分隔。  |
| 主分片数         | 设置索引的主分片数。                       |
| 副本数           | 设置索引的副本数。                      |
| Translog（可选） | 选择**异步刷盘**或**同步刷盘**。           |
| 刷新间隔（可选） | 设置索引模板刷新的间隔时间，单位为秒。            |
| 压缩方式（可选） | 选择**速度优先**、**压缩比优先**。          |
| 索引别名         | 设置索引的别名。                           |
| 索引设置（可选） | 以 JSON 格式手动修改索引模板内容，将覆盖以上参数的值。 |

**UES 7.10 参考模板:**

```
{
  "order": 5,
  "index_patterns": [
    "log-*"
  ],
  "settings": {
    "index": {
      "codec": "best_compression",
      "refresh_interval": "10s",
      "number_of_shards": "5",
      "translog": {
        "durability": "async"
      },
      "number_of_replicas": "1"
    }
  },
  "mappings": {
    "dynamic_templates": [
      {
        "strings_as_keyword": {
          "mapping": {
            "ignore_above": 1024,
            "type": "keyword"
          },
          "match_mapping_type": "string"
        }
      }
    ]
  },
  "aliases": {
    "logs-current": {}
  }
}
```

## 编辑索引模板

![image](/images/operate/index_template_2.png)

1. 在索引管理页面，选择索引模板页签，然后在目标索引模板的操作列下，单击编辑。
2. 在编辑索引模板面板中，根据需要修改各参数，然后单击确定。

## 删除索引模板

1. 在索引管理页面，选择索引模板页签，然后在目标索引模板的操作列下，单击删除。
2. 在弹出的对话框中，确认索引模板信息，输入索引模板名称，然后单击删除。
